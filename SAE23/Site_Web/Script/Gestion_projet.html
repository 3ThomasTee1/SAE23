Difficultés rencontréSolution trouvé
Choix des parties qui composent une page web (margin, padding..) A mesure que j'avançais sur mon portfolio, j'ai compris comment fonctionnaient ces différents blocs afin de les utiliser à bon escient.
Apprentissage des commandes de style De la même façon, les commandes de style deviennent plus parlante au fur et à mesure que je les utilise.
Le Responsive Web Design. C'est une étape longue est fastidieuse Il faut le faire au fur est à mesure que l'on crée notre site web pour arriver à s'y retrouver dans son fichier css. Commenter mon fichier css a été d'une grande aide.
J'ai voulu essayer de rendre mon image multi-cliquable responsive mais, je n'ai pas trouvé la solution en HTML. Pas de solution
Je n'ai pas inséré de balise audio car, je n'ai pas trouvé de site de convertisseur mp3 assez sécurisé pour télécharger mes musiques. Pas de solution
